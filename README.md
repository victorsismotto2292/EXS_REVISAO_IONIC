* Enunciado dos Exercícios *
  
- 1.Simulador de compra de tênis
- Crie uma tela onde o usuário informe o preço de um tênis e escolha, via ion-radio, um cupom
de desconto (10%, 20%, 30% ou “sem cupom”). Ao confirmar, exiba um alert com:
- • preço final
- • valor economizado
- • mensagem personalizada (ex: “Boa escolha!” se desconto alto)
- 
2. Setup gamer (orçamento)
Crie uma aplicação onde o usuário informe seu orçamento total e selecione, via ion-radio, um
tipo de setup (Básico, Intermediário, Avançado), cada um com um custo estimado. Ao
confirmar, exiba um alert mostrando:
• se o orçamento é suficiente
• diferença de valor (quanto falta ou sobra)
3.Divisão de conta de delivery
Monte uma tela onde o usuário informa o valor total de um pedido (ex: pizza ou hambúrguer)
e quantas pessoas vão dividir. Depois, escolha via ion-radio a taxa adicional (frete grátis,
+10%, +15%). Ao finalizar, exiba um alert com:
• valor total com taxa
• valor que cada pessoa deve pagar
• mensagem divertida (ex: “Já pode cobrar no Pix!”)
