Enunciado dos Exercícios

1. Simulador de Compra de Tênis
Crie uma interface de checkout simples para aplicação de descontos.

Requisitos:

Entrada: Preço do tênis (input numérico).

Seleção: Escolha do cupom via ion-radio:

10%, 20%, 30% ou "Sem cupom".

Ação: Botão de confirmação que dispara um alert.

Exibição no Alert:

Preço final calculado.

Valor total economizado.

Mensagem personalizada (Ex: "Boa escolha!" para descontos altos).

2. Setup Gamer (Orçamento)
Uma ferramenta para validar se o sonho do PC novo cabe no bolso do usuário.

Requisitos:

Entrada: Orçamento total disponível.

Seleção: Tipo de setup via ion-radio:

Básico: Custo X

Intermediário: Custo Y

Avançado: Custo Z

Ação: Botão de confirmação que dispara um alert.

Exibição no Alert:

Confirmação se o orçamento é suficiente.

Diferença de valor (quanto sobra ou quanto falta para atingir a meta).

3. Divisão de Conta de Delivery
Calculadora para facilitar o rateio de pedidos entre amigos.

Requisitos:

Entrada: Valor total do pedido e quantidade de pessoas.

Seleção: Taxa de entrega via ion-radio:

Frete grátis, +10% ou +15%.

Ação: Botão de confirmação que dispara um alert.

Exibição no Alert:

Valor total (pedido + taxa).

Valor individual por pessoa.

Mensagem divertida (Ex: "Já pode cobrar no Pix!").
